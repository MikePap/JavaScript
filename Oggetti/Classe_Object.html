La classe Object

La classe Object è la classe base per per tutte le classi JavaScript da cui tutti gli oggetti JavaScript derivano.
Tutte le proprietà e metodi della classe Object sono presenti in tutte le altre classi.
La classe Object ha le seguenti proprietà:

  • constructor: per la classe Object, punta alla funzione nativa Object
  • prototype: per tutte le classi, restituisce un'istanza di Object per default
var myObj = new Object("foo");
