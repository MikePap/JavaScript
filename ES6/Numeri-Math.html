ES6 ha introdotto nuovi metodi interni sui numeri e su Math

Numeri

Number.isFinite

Number.isFinite(Infinity)				// false
Number.isFinite(-Infinity)				// false
Number.isFinite(NaN)					// false
Number.isFinite(123)					// true
Number.isFinite('123')					// false
mentre	
isFinite('123')						// true

Number.isInteger(number)

Number.isInteger(-14)					// true
Number.isInteger(25)					// true
Number.isInteger(25.4)					// false
Number.isInteger('25')					// false
Number.isInteger(NaN)					// false
Number.isInteger(Infinity)				// false

Math

Math.sign(x)

Restituisce il segno di un numero a seconda che esso sia negativo o positivo

Math.sign(-8)						// -1
Math.sign(8)						// 1
Math.sign(0)						// 0
Math.sign(NaN)						// NaN
Math.sign(Infinity)					// 1
Math.sign(-Infinity)					// -1

Math.trunc(x)

Rimuove la frazione decimanle da un numero 

Math.trunc(3.1)						// 3
Math.trunc(3.8)						// 3
Math.trunc(-3.1)					// -3
Math.trunc(-3.8)					// -3

Math.cbrt(x)

Restituisce la radiuce cubica di un numero

Math.cbrt(8)						// 2

Altri metodi sono:

Math.expm1(x)
Math.log1p(x)
Math.log2(x)
Math.log10(x)
Math.fround(x)
Math.imul(x, y)
