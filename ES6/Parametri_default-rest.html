Parametri di default

ES6 ha introdotto, come altri linguaggi di programmazione, la possibiltà di definire dei parametri di default per le funzioni, sebbene in ES5 si possono definire in un'altro modo 

ES5
function sayHi(name, place){
	name = name || 'Ravi';
	place = place || 'Tokio';
	console.log("Hello, " + name +" from " + place + "!");
}

ES6
function sayHi(name = 'Ravi', place = 'Tokio'){
	console.log("Hello, " + name +" from " + place + "!");
}

E' inoltre possibile definire un parametro che si basi su un'altro parametro, come nel seguente esempio 

function sayHi(name = 'Ravi', place = 'Tokio', message = "No additional message for " + name){
	console.log("Hello, " + name +" from " + place + "! (" + message + ")");
}

Come si può notare il parametro name è usato dal parametro message

Parametri Rest

Questo metodo da la possibilità di invocare una funzione con svariati parametri ma definendola con un semplice nome preceduto da tre puntini 

function addNumbers(...numbers){
	var total = 0;
	numbers.forEach(number => {
		total += number;
	});
	console.log(total);
}

addNumbers(10,25,35,10);			// 80

In verità si possono passare anche altri parametri oltre al rest ma ad ogni modo è consigliabile definirlo come ultimo parametro
