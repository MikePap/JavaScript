Stringhe

ES6 ha introdotto nuovi metodi interni per le Stringhe

startsWith(), endsWith(), includes() e repeat()

'hello'.startsWith('hell')				// true
'hello'.endsWith('ello')				// true
'hello'.includes('ell')					// true
'doo '.repeat(3)					// doo doo doo

String padding

String.prototype.padStart

'5'.padStart(10)			// '          5'
'5'.padStart(10, '=*')			// '=*=*=*=*=5'

String.prototype.padEnd

'5'.padEnd(10)				// '5         '
'5'.padEnd(10, '=*')			// '5=*=*=*=*='

Template strings

E' un modo di rappresentare o se vogliamo di richiamare le variabili attraverso dei simboli che sono il dollaro ($) e le parentesi graffe {} e racchiudere le espressioni tra virgolette contrarie `` (In windows: Alt + 96 --- in Ubuntu: AltGr + ' ) 

var x = 1;
var y = 2;
var somma = `${ x } + ${ y } = ${ x + y}`;

console.log(somma);						// "1 + 2 = 3"	
	
è un altro modo di scrivere	

var somma = x+ ' + ' +y+ ' = ' +(x+y);

String.trim(), String.trimStart() e String.trimEnd()

Sono metodi che permettono di rimuovere eventuali spazi vuoti prima e dopo la stringa 


let salutone = "   Hello World    ";
console.log(salutone.length);			// 18

trimStart: rimuove gli spazi vuoti presenti all'inizio della stringa
let saluto = salutone.trimStart();
console.log(saluto);				// "Hello world!   "

trimEnd: rimuove gli spazi vuoti presenti dopo la fine della stringa
let saluto = salutone.trimEnd();
console.log(saluto);				// "   Hello world!"

trim: rimuove gli spazi vuoti presenti sia prima che dopo la stringa
let saluto = salutone.trim();
console.log(saluto);				// "Hello world!"
console.log(saluto.length);			// 11
