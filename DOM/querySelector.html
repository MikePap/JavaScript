document.querySelector

element = document.querySelector(selectors);
Individua l'elemento specificato dal selettore. Supportato da IE 8 

var div = document.querySelector('#divo');			// individua l'elemento div id='divo'
var box = document.querySelector('div.box');			// individua l'elemento div class='box'
var span = document.querySelector('div.box > span');		// individua l'elemento span all'interno del div di classe box


Se, per esempio, si hanno due paragrafi che hanno la stessa classe: 

<p class="myclass">primo paragrafo</p> 
<p class="myclass">secondo paragrafo</p> 

var el = document.querySelector(".myclass");
console.log(el.textContent);				// restituisce: primo paragrafo
console.log(el.length);					// restituisce: undefined

paragrafo

Individuare un elemento dagli attributi 
<div class='className' attributo='valoreAttributo' > div con attributi </div>  
var a = document.querySelectorAll("div[attributo]")[0];
var a = document.querySelectorAll("div.className")[0];

Individuare un elemento con due classi 
<div class='classe className' > div con due classi </div>  
var elem = document.querySelectorAll("div.classe.classeName")[0];

Liste pre-configurate

Il documento HTML contiene delle liste pre-configurate di tipo array

document.all: tutti gli elementi del documento HTML
document.forms: tutti gli elementi <form>
document.images: tutti gli elementi <img>
document.links: tutti gli elementi <a>
document.scripts: tutti gli elementi <script>
document.styleSheets: tutti gli oggetti <link> e <style>
