Cambiare gli stili

Tramite il DOM si può accedere a quasi ogni proprietà della pagina compresi i suoi vari elementi estetici
Attracerso la proprietà style si può modificare ogni caratteristica dell'aspetto di un elemento (colore,altezza, testo, bordo ecc..) ma questa è una pratica sconsigliata, perchè per lo style è meglio usare i CSS
Es: (modifica dello sfondo e del colore del testo di un documento)
var body = document.getElementsByTagName("body")[0];
body.style.backgroundColor = "#000000";
body.style.color = "#FFFFFF"; questo script va inserito nel tag body

Gestione stili tramite la classe Core

Controllare se un elemento contiene una determinata classe

Per controllare se un elemento contiene una determinata classe si può fare uso della funzione Core.hasClass della libreria (file) core.js nella cartella book.

questo è un testo con classe='nota' e id='classe'
var trova_classe = document.getElementById("classe");
if(Core.hasClass(trova_classe, "nota"))
{ esegui codice }
Se la classe non esiste restituisce false(non da risultato)

Aggiunta di una classe

Per aggiungere una classe ad un elemento si può utilizzare la funzione Core.addClass della libreria (file) core.js nella cartella book

questo è un paragrafo con classe='nota' e id='ad_classe'
var paragrafo = document.getElementById("ad_classe");
Core.addClass(paragrafo, "grassetto"); ora al paragrafo verrà aggiunta una nuova classe grassetto che naturalmente deve stare nel CSS

Eliminazione di una classe

Per eliminare una classe da un elemento si può uyilizzare la funzione Core.removeClass della libreria (file) core.js nella cartella book

questo è un paragrafo con classe='nota' e id='remove_classe'
var paragrafo = document.getElementById("remove_classe");
Core.removeClass(paragrafo, "nota"); così al paragrafo viene rimossa la classe nota
