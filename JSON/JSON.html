JSON

JSON sta per Notazione Oggetti JavaScript. In altre parole esso usa JavaScript per una rappresentazione strutturale dei dati. In definitiva è un formato di dati e non un linguaggio di programmazione.
La sintassi di JSON permette la rappresentazione di tre tipi di valori:

Simple Values
stringhe, numeri, boleani, null. La parola 'undefined' non è supportata.
Objects
coppie di valori rappresentate da una chiave e un valore corripspondente. Ogni valore può a sua volta essere un object o un oggetto più complesso
Arrays
rappresenta un ordinata lista di valori che sono accessibile da un indice numerico. I valori possono essere Simple Values, object o altri arrays

Simple Values

// un qualsiasi numero è espresso cosi com'è senza doppie o singole virgolette 
8
// per la rappresentazione di una stringa bisogna usare sempre le virgolette doppie
"Hello world"

Objects

In JavaScript un esempio di rappresentazione di un oggetto può essere la seguente: 

var tutorial = 
{ 
	name: "JavaScript", 
	page: 9 
};
mentre JSON lo stesso oggetto lo rappresenta così: 
{ 
	"name": "JavaScript", 
	"page": 9 
}
dove le virgolette doppie sono obbligatorie per rappresentare il nome della proprietà; una proprietà è separata dall'altra da una virgola tranne che per l'ultima. Nell'esempio seguente vediamo come un valore può essere rappresentato da un valore semplice oppure complesso: 
{
	"name": "JavaScript", 
	"page": 9, 
	"topic":{
			"name":    "Data Type", 
			"content": "Number" 
		} 
}

Arrays

Gli arrays sono rappresentati in JSON usando lo stesso tipo di sintassi come in JavaScript: 

var values = [1, "JavaScript", true];		// array in JavaScript

[1, "JavaScript", true]				// array rappresentato in JSON

Arrays e Objects possono essere usati insieme per rappresentare una serie più complessa di dati: 

[
	{ 
		"title": "JavaScript", 
		"authors": ["J"],
		edition: 3,
		year: 2011
	}, 
	{ 
		"title": "HTML", 
		"authors": [ "H" ],
		edition: 2,
		year: 2009
	},
	{ 
		"title": "Ajax", 
		"authors": ["X", "Y", "Z" ], 
		edition: 2, 
		year: 2008
	},
	{ 
		"title": "CSS", 
		"authors": ["A", "B", "C" ], 
		edition: 1, 
		year: 2007
	},
	{ 
		"title": "Java", 
		"authors": ["C"],
		edition: 1,
		year: 2006
	} 
]
