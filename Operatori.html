Operatori

! (Operatore logico NOT)

Espressione Restituisce
Se l'operando è un oggetto false
Se l'operando è un numero diverso da zero false
Se l'operando non è definito Un errore è capitato
Se l'operando è un numero true
Se l'operando è NULL true
Se l'operando è NaN true

!= (non uguale) e !== (Non identico)

!= restituisce true se gli operandi non sono uguali.
!== restituisce true se gli operandi non sono uguali e nemmeno dello stesso tipo

Operatore condizionale ternario

L'operatore condizionale ternario è l'unico operatore JavaScript che necessità di tre operandi. Questo operatore è frequentemente usato al posto del comando «if» per la sua sintassi concisa e perché fornisce direttamente un espressione valutabile. 

condizione ? espressione1 : espressione2
condizione: Un espressione booleana (che viene valutata in vero o falso).
espressione1, espressione2: Espressione di qualunque tipo.
Se la condizione è vera sarà restituito "espressione1" altrimenti sarà restituito "espressione2"
let prezzo;
let chePrezzo = "Il prezzo è: " + (prezzo ? "$2.00" : "$10.00");		// $10.00

Anche definendo «let prezzo = false» abbiamo lo stesso risultato; mentre se "prezzo" è definita oppure è true

let prezzo = true;
let chePrezzo = "Il prezzo è: " + (prezzo ? "$2.00" : "$10.00");		// $2.00

var elvisLives = Math.PI > 4 ? "Yep" : "Nope";					// Nope

Con l'utilizzo di operatori ternari multipli

var firstCheck = false,
    secondCheck = false,
    access = firstCheck ? "Access denied" : secondCheck ? "Access denied" : "Access granted"

console.log(access);								// Access granted

E' possibile assegnare alle espressioni più valori mettendoli tra parentesi.
Nell'esempio seguente nelle due espressioni sono definiti due processi separati da una virgola e racchiusi in una parentesi 

var stop = false, age = 17;

age > 18 ? (
    alert("OK, you can go."),
    location.assign("continue.html")
) : (
    stop = true,
    alert("Sorry, you are much too young!")
);

L'esempio seguente mostra l'uso dell'operatore ternario all'interno di una funzione 

Con l'uso degli "if"	
function opTernario(valore){
	if(valore){
		return 'OK';
	}else{
		return 'NO';
	}
}

Con l'operaore ternario
function opTernario(valore){
	return valore ? 'OK' : 'NO';
}

console.log(opTernario());				// NO
console.log(opTernario('bene'));			// OK

Operatore Modulo %

$a % $b = risultato

L'operatore modulo, o resto, divide l'argomento $a per l'argomento $b e restituisce il resto come risultato.
Se $a=1000 e $b=255 il resto sarà 235. In altre parole avremo che 1000 = (255*3) + 235

Una cosa che capita è quella di rilevare i pari e i dispari in una iterazione.
Utilizzando questo modulo lo possiamo fare molto semplicemente mettendo in relazione l'iteratore e il numero 2.
Ciò a dire che se il numero dell'iteratore è dispari, in relazione a 2, si avrà sempre come risultato uno, mentre se l'iteratore è un numero pari si otterrà lo zero.
In altre parole, qualsiasi numero dispari in relazione con 2 da come risultato uno. Mentre qualsiasi numero pari da zero. 


for(var i=0; i < 10; i++) {
	if(i % 2 == 1) {					// 1 3 5 7 9
		console.log(i);
	} 
}

mentre

for(var i=0; i < 10; i++) {
	if(i % 2 == 0) {
		console.log(i);					// 0 2 4 6 8
	}
}
