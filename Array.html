Array Object

L'array è un tipo speciale dell'oggetto Object() contenente una lista ordinata di valori. Ogni valore può essere di qualsiasi tipo ed a ognuno di questi è collegato una chiave numerica (non ordinata). La prima chiave ha valore 0 

var variable = new Array()
var variable = new Array(int)
var variable = new Array("arg1", ..., "argN")
var variable = ["elemento1", "elemento2", "elemneto3"];

JavaScript non ha array associativi, come per esempio PHP, ma per tale uso si possono utilizzare gli oggetti

var myObject = { 
	'blue': 'blue',
	'green': 'green',
	'orange': 'orange',
	'red': 'red'
};

Se all'array viene passato un solo valore numerico tale valore verrà usato per impostare la lunghezza dell'array: 

var arrayL = new Array(20); 
console.log(arrayL.length);		// restituisce 20 (è la nuova lunghezza dell'array)

		però con le parentesi quadre le cose cambiano

var arrayL = [20];			
console.log(arrayL.length);		// restituisce 1

Array length

La proprietà length degli array è read/write e quindi può essere cambiata in qualsiasi momento.
Se length viene aumentato dinamicamente un numero di elementi viene aggiunto alla fine dell'array a cui è assegnato un valore indefinito. Se la length viene ridotta elementi alla fine dell'array vengono persi (vedere l'esempio) 

var cats = [];  
cats[30] = ['Dusty'];  
alert(cats.length);					 // 31 

var cats = ['Dusty', 'Misty', 'Twiggy'];  
alert(cats.length);					 // 3
      
cats.length = 2;  
alert(cats.length);					// "Dusty,Misty" (Twiggy è stato rimosso) 
      
cats.length = 0;  
alert(cats.length);					// 0 (l'array è di nuovo vuoto)

var numeri = [1, 2, 10, 21];

Ordinamento di caratteri non-ASCII col metodo localeCompare

In caso di ordinamento di caratteri non-ASCII tipo caretteri accentati (e, é, è, a, ä, ecc.. ) bisogna usare il metodo String.localeCompare 

var paroleAccentate = ['réservé', 'premier', 'cliché', 'communiqué', 'café', 'adieu'];

paroleAccentate.sort(function (a, b) {

for(var i=0; i < paroleAccentate.length; i++){

Array.prototype.slice()

Metodo che permette di estrarre elementi da un array. In altre parole crea un nuovo array. 

arr.slice([begin[, end]])

Il parametro 'begin' indica l'inizio dell'estrazione e 'end' dove finisce. Se 'begin' è omesso l'estrazione parte dall'indice zero; 

var frutta = ['mela', 'pera', 'banana', 'mango', 'melone'];
var rimuoviFrutti = frutta.slice(1, 3);

console.log(frutta);					// Array [ "mela", "pera", "banana", "mango", "melone" ]
console.log(rimuoviFrutti);				// ["pera", "banana"]
console.log(frutta.length);				// 5 (non altera la 'lenght' dell'array iniziale perchè crea un nuovo array ...)
console.log(rimuoviFrutti.length);			// 2 (... ecco dimostrato che 'rimuoviFrutti' è un nuovo array)
var dalTerzo = frutta.slice(2);				// ["banana", "mango", "melone"] (senza secondo parametro l'estrazione si potrae fino alla fine) 
var primiDue = frutta.slice(null, 2);			// ["mela", "pera"] (senza primo parametro l'estrazione parte da zero)

Il metodo slice permette di convertire oggetti di tipo array in veri array. E' il caso, ad esempio, di elementi del DOM (tags). La funzione potrebbe essere la seguente 

function lista() {
	return Array.prototype.slice.call(arguments, 0);
}
var lista1 = lista(1, 2, 3);

console.log(lista1);						// [1, 2, 3]

All'interno c'è un polyfill che risolve una incompatilità dei browser sull'uso di Array.prototype.slice con oggetti del DOM

Array.prototype.splice()

Il metodo può far cambiare il contenuto di un'array sia permettendo l'aggiunta che la rimozione di elementi 

splice(start, delete, arg1..argN)

Il parametro delete rappresenta il numero di valori da eliminare; start rappresenta la partenza; arg1 e argN sono nuovi elementi (valori) da aggiungere all'array partendo dalla posizione indicata in start. Se delete è uguale a 0 nessun elemento viene eliminato; se delete non viene specificato tutti gli elementi dell'array vengono eliminati.
La length finale viene influenzata dai cambiamenti. 

var frutta = [ "mela", "pera", "banana", "mango", "melone" ];

var t1 = frutta.splice(2, 0, 'ciliege');	 dopo il secondo elemento aggiunge  'ciliege' senza eliminare nulla	 
console.log(frutta);				// ["mela", "pera", "ciliege", "banana", "mango", "melone"] 
console.log(t1);				// [] 
console.log(frutta.length);			// 6 (è stata modificata e non è più 5) 

var t2 = frutta.splice(2, 2);			dopo il secondo ne elimina due
console.log(frutta);				// ["mela", "pera", "mango", "melone"] 
console.log(t2);				// ["ciliege", "banana"] 
console.log(frutta.length);			// 4 (la length viene ulteriormente modificata) 

var t3 = frutta.splice(2, 2, 'pesca');		ne elimina 2 dopo il secondo e aggiunge "pesca"		
console.log(frutta);				// [ "mela", "pera", "pesca"
console.log(t3);				// [ "mango", "melone" ] 
console.log(frutta.length);			// 3 (la length viene ulteriormente modificata) 

var t4 = frutta.splice(2);			senza il parametro 'delete' vengono eliminati tutti a partire dal secondo
console.log(frutta);				// [ "mela", "pera"]
console.log(t4);				// [ "pesca" ] 
console.log(frutta.length);			// 2
